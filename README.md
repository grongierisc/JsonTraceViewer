![img](https://github.com/grongierisc/JsonTraceViewer/blob/master/json-message-trace-metadata.png?raw=true)

## How to Run

1. Open the terminal in this directory and run:

```
docker-compose build
```

1. Run the IRIS container:

```
docker-compose up -d
```
